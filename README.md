# Pawps PH — Montalban

Pet supply ERP for inventory, orders, suppliers, delivery, and reports.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Database

```bash
npm run db:migrate
```
